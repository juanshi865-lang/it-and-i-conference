# IT&I 2024 Conference Website

Static website for:

IT&I 2024 - XI International Scientific Conference "Information Technology and Implementation"

## Pages

- `index.html`: conference home page
- `cfp.html`: call for papers
- `submit.html`: author submission draft form
- `proceedings.html`: searchable digital proceedings
- `paper.html`: dynamic paper detail page
- `committee.html`: committee page with verified-member placeholders
- `publisher.html`: editor publication desk

## Publishing Papers

The site currently includes one published paper record:

- Jing Li, "Adaptive Strategy Optimization for Long-Horizon Multi-Channel B2B Campaigns"

1. Open `publisher.html`.
2. Add the accepted paper metadata.
3. Add or create a web article page under `papers/`.
4. Preview it in `proceedings.html`.
5. Click `Export papers.js`.
6. Replace `assets/papers.js` with the exported file before deploying.

The site publishes article content as web-native pages with proceedings metadata and full text.
